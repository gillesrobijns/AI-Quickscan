# Ethias — AI-Accelerated OpEx Quickscan (Interactive Demo)

A **self-contained, single-file** interactive dashboard demonstrating Nova Reperta's
AI-Accelerated OpEx Quickscan, populated with **illustrative dummy data** modelled on a
Belgian P&C insurer value chain (Claims, Underwriting, Distribution, Actuarial,
Reinsurance, Investments, …).

> ⚠️ **Demo only.** All figures, teams, names and findings are fictional and for
> illustration. This is **not** a real Ethias assessment.

## Use it

Just open **`index.html`** in any modern browser. No build step, no server, no external
files — everything (data, styles, scripts) is inlined.

### Host on GitHub Pages
1. Create a repo and add `index.html` (and this README).
2. Settings → Pages → Build from branch → `main` / root.
3. Open the published URL. Done.

## What's inside
- Executive Summary, Transformation Roadmap, Benefits Overview
- All Opportunities + Team Deep-Dives (16 teams / 468 FTE, illustrative)
- Light / dark theme toggle

Internal project tools (Workspace, configurator, export, guide) are intentionally
disabled for this external demo build.
