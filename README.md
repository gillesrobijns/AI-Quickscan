# AI-Accelerated OpEx Quickscan — Interactive Demo

**▶ Live demo: https://gillesrobijns.github.io/AI-Quickscan/**

A self-contained, single-file interactive dashboard that demonstrates Nova Reperta's
AI-Accelerated OpEx Quickscan. It is populated with **illustrative dummy data** modelled
on a Belgian P&C insurer value chain (Claims, Underwriting, Distribution, Actuarial,
Reinsurance, Investments, and corporate functions).

> ⚠️ **Demo only — not a real assessment.** Every team, name, figure and finding in this
> dashboard is fictional and for illustration. It does not represent any real client.

## Use it

- **Online:** open the live link above — works in any modern browser, nothing to install.
- **Offline:** download `index.html` and double-click it. Everything (data, styling,
  scripts) is inlined in that one file, so it runs with no server and no external
  dependencies.

## What's inside

- **Executive Summary** — headline findings, maturity assessment, key priorities.
- **Transformation Roadmap** — sequenced two-year view across improvement levers.
- **Benefits Overview** — value potential and FTE impact across the portfolio.
- **All Opportunities** — the full opportunity register, filterable.
- **Team Deep-Dives** — per-team diagnostics for 16 teams (~468 FTE, illustrative).
- Light / dark theme toggle.

Internal project-delivery tools (workspace, configurator, export, methodology guide) are
intentionally disabled in this external demo build.
